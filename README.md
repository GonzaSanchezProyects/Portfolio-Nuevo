# Portfolio Personal - Gonzalo Sanchez

Un portfolio web interactivo y de alto rendimiento construido para destacar mi experiencia como Desarrollador Fullstack y especialista en automatización. Este proyecto no solo actúa como un currículum digital, sino como una demostración técnica de mis habilidades en desarrollo frontend, manejo de estado global y diseño UI/UX avanzado.

## Características Principales

* **Arquitectura Bilingüe:** Sistema de internacionalización (Español/Inglés) implementado de forma nativa utilizando React Context API (`AppContext`), permitiendo cambiar el idioma en tiempo real sin recargar la página.
* **Efectos Visuales Avanzados:** Implementación de componentes interactivos personalizados, incluyendo un efecto "Spotlight" dinámico que sigue el cursor, interfaces basadas en Glassmorphism y carruseles cinemáticos para la galería de proyectos.
* **Componentización Modular:** Estructura de código altamente escalable. Componentes clave aislados (como `TechTicker`, `Projects`, `Gencore`) con sus propios módulos CSS (`.module.css`) para evitar colisiones de estilos.
* **Diseño 100% Responsivo:** Adaptación fluida a dispositivos móviles mediante media queries precisas, alterando la interacción (por ejemplo, reemplazando el comportamiento *hover* por botones táctiles en pantallas pequeñas).

## Stack Tecnológico

* **Core:** React 18, Vite
* **Enrutamiento:** React Router DOM
* **Estilos:** CSS Modules, variables globales de CSS (`variables.css`)
* **Iconografía:** react-icons (FontAwesome & Simple Icons) y Lucide React
